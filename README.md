# Discord Embed Studio

A minimal, browser-based tool to create and preview Discord embeds and send them via webhooks.

## Features
- Embed builder (title, description, footer, color)
- Live preview (Discord-style)
- Random template generator
- Local save (localStorage)
- 30-second cooldown to reduce spam risk

## How to use
1.Open the Github-Pages [link](https://lucbre-2009.github.io/webhook-sender/)
2.Use it :)

## Safety note
This tool does not include backend protection.  
Anyone with a webhook URL can post to it.  
Use only your own webhooks and do not share them publicly.
The Tool is not made for illegal activities.

## Tech
- HTML
- CSS
- JavaScript (vanilla)

## Limitations
- No authentication
- No server-side protection
- Cooldown is client-side only

## Goal
A lightweight, fast embed editor that works on PC and Mobile devices without setup.
